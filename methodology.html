Methodology

The machine learning pipeline and evaluation framework behind habitability prediction

Hierarchical Machine Learning Pipeline

A sophisticated two-stage workflow designed to handle class imbalance and high-precision classification

Our research utilizes a Two-Stage Hierarchical XGBoost approach. Stage 1 separates candidates from non-habitable noise, while Stage 2 performs granular refinement using SMOTE oversampling to identify true Earth-like candidates.

Machine Learning Models

Four classification algorithms evaluated for habitability prediction

📈

Logistic Regression

Linear classifier used as a baseline model. Provides interpretable decision boundaries and probability estimates for each habitability class.

Baseline
🌲

Decision Tree

Tree-based classifier that creates interpretable if-then rules. Useful for understanding which parameter thresholds separate habitability classes.

Interpretable
🌳

Random Forest

Ensemble of decision trees that reduces overfitting through bagging. Provides robust predictions and built-in feature importance ranking.

Ensemble
⚡

Hierarchical XGBoost

The lead model in our research. Uses two dedicated boosting stages to achieve near-perfect precision (97%) and recall (95%) on the imbalanced exoplanet dataset.

State-of-the-Art

Preprocessing Techniques

Data cleaning and transformation methods applied before model training

⚖️

Feature Scaling

StandardScaler normalization ensures all features contribute equally to model training regardless of their original magnitude or unit of measurement.

🔧

Missing Value Handling

Imputation strategies replace missing values using statistical measures (mean/median) or model-based predictions to preserve dataset integrity.

⚖️

Class Imbalance

SMOTE oversampling and class weighting techniques address the natural imbalance between habitable and non-habitable planet classifications.

Evaluation Metrics

Standard metrics used to assess model performance

Accuracy

Ratio of correctly classified planets to total predictions

Precision

How many predicted habitable planets are truly habitable

Recall

How many actual habitable planets were correctly identified

F1 Score

Harmonic mean of precision and recall for balanced evaluation

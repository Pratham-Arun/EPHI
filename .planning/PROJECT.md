# EPHI – Exoplanet Habitability Index

## Vision
A multi-page web platform demonstrating exoplanet habitability analysis using planetary and stellar parameters. The platform provides an interactive educational interface for exploring exoplanet data, visualizing planetary orbits, and estimating habitability using simplified heuristic rules.

## Stack
- **Core:** HTML5, CSS3, JavaScript
- **Visualization:** Chart.js
- **Design:** Space-themed dark UI with cosmic backgrounds, neon accents, CSS animations

## Color Palette
- Space Black: `#0b0f1a`
- Deep Blue: `#111827`
- Accent Cyan: `#38bdf8`
- Highlight Blue: `#60a5fa`
- Text White: `#f9fafb`

## Pages
1. **Home** — Hero with rotating 3D planet, project intro
2. **About** — Scientific background on exoplanets
3. **Methodology** — ML workflow and research approach
4. **Dataset** — Feature descriptions & data sources
5. **Results** — Chart.js visualizations of analysis
6. **Explorer** — Interactive habitability predictor with heuristic model

## Strategy
- Front-end only (no backend)
- Spec-driven development via GSD
- Habitability prediction in simulation/demo mode

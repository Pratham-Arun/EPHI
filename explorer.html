How the Explorer Works

Understanding the simplified habitability scoring process

1️⃣

Parameter Input

You provide four key planetary parameters: orbital distance, stellar temperature, planet radius, and planet mass. These represent the most influential factors in habitability determination.

2️⃣

Heuristic Calculation

The system computes derived features (insolation flux, equilibrium temperature, surface gravity) and evaluates them against Earth-like ranges using weighted scoring functions.

3️⃣

Classification Output

Based on the composite score, the planet is classified as Habitable, Potentially Habitable, or Non-Habitable, along with a simulated confidence percentage.
